# `ui-ux-pro-max`

## Purpose

Provides broad UI/UX design intelligence, including styles, palettes, font pairings, accessibility, animation, layout, typography, dashboards, landing pages, charts, and stack-specific guidance.

## Use When

- Designing or reviewing UI/UX work.
- Creating a product experience baseline.
- Choosing visual direction, palettes, typography, layout, or responsive behavior.
- Auditing accessibility and interaction quality.
- Implementing dashboards, admin panels, SaaS pages, portfolios, e-commerce, mobile apps, or charts.

## Best Paired Commands

- `/sdd-constitution` for system design.
- `/sdd-specify` for UI requirements and design.
- `/sdd-execute` for implementation guidance.
- `/sdd-test` for UX checks.
- `/sdd-validate` for UI conformance.
